# СВИТОК ЮРИСТ

## Описание

Веб-приложение для демонстрации AI-анализа юридических документов.

Функциональность:

* выбор договора;
* имитация процесса анализа;
* отображение выявленных рисков;
* краткое заключение по документу;
* автозаполнение реквизитов;
* визуализация результатов анализа.

## Технологии

* React
* TypeScript
* Vite
* TailwindCSS
* Framer Motion

## Запуск проекта

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```
